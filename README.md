## ファイル構成
    - main.py
        + 実行用ファイルです。アイテムの分布や実行時間の測定などを行っています。
    - problem.py
        + ビンパッキング問題とその双対問題を解くための関数が定義されています。
    - distribution.py
        + アイテムの分布を決める関数を定義しています。

## 実行時
    - Fish Script である main.fish にてビンのサイズ b とアイテムの個数の最大値 n を記入してください。
    - main.py の dis_n となっている関数を任意の値に書き換えてください。
    - b のサイズでアイテムの個数を 10〜n の範囲で 10 個ずつ増やしながらビンパッキング問題を列生成法で解きます。
    - 実行時間が output.txt に出力されます。
